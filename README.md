# AI競馬コンセンサス

複数の競馬AI予想サイトの評価を統合し、買い目ではなく「AI支持率ランキング」を表示するプロトタイプです。

## 使い方

`index.html` をブラウザで開くと動きます。サーバーや追加インストールは不要です。

土日稼働の確認をしたい場合は、URL末尾に `?demoTime=2026-06-20T10:00:00+09:00` を付けると、テスト用に土曜午前として表示できます。

地方競馬版は別アプリとして `chihou/index.html` に分けています。GitHub Pagesでは `/chihou/` を開くと地方競馬版を表示できます。

GitHub Pagesで公開する場合は、`DEPLOY_GITHUB_PAGES.md` の手順に沿ってアップロードしてください。

## 実装済み

- JRAレース一覧の表示
- JRA 10競馬場対応（札幌・函館・福島・新潟・東京・中山・中京・京都・阪神・小倉）
- 当日開催レースだけを表示
- 土日 09:00〜17:00 のみ稼働する取得判定
- 10サイトを参照対象にした支持率計算
- AI支持率ランキング
- 馬ごとのサイト別評価
- オッズとAI順位の乖離表示
- スマホ向けのコンパクトランキング表示
- 馬複・馬単・三連複・三連単の推奨買い目候補
- 週次成績データを使ったサイト重みの自動反映
- 別ページでのサイトごとの重み確認
- 別ページでの週次成績確認
- 最終更新時刻、取得サイト数、未取得サイト一覧
- GitHub Actionsによる土日 09:00〜17:00 の10分おきデータ更新

## 自動更新

GitHub Actionsの `Update JRA race data` が、土日のJRA開催時間帯に10分おきで `docs/data/jra-races.json` を更新します。

外部の実データJSONを使う場合は、GitHub Secretsに `JRA_RACES_JSON_URL` を設定します。未設定の場合は、アプリ内のレース雛形から公開用データを生成します。
